Exercice – Créer la logique de jeu

Une fois Codespaces configuré, votre objectif est de développer le mini-jeu de console en Python. Vous devez créer la logique de jeu en suivant les spécifications et utiliser GitHub Copilot pour créer les méthodes.

Caractéristique

Règles du jeu :

  • La pierre bat les ciseaux (elle les casse).
  • Les ciseaux battent la feuille (ils la coupent).
  • La feuille bat la pierre (elle l’enveloppe).
  • Le mini-jeu est multijoueur et l’ordinateur joue le rôle de votre adversaire et choisit un élément aléatoire dans la liste des éléments.

Interaction avec le joueur :

  • La console est utilisée pour interagir avec le joueur.
  • Le joueur peut choisir l’une des trois options : rock, paper ou scissors.
  • Le joueur peut choisir de rejouer ou non.
  • Le joueur doit être averti s’il entre une option non valide.
  • Le score du joueur s’affiche à la fin de la partie.

Validation de l’entrée utilisateur :

  • À chaque tour, le joueur doit entrer l’une des options de la liste et être informé s’il a gagné, a perdu ou est à égalité avec son adversaire.
  • Le mini-jeu doit gérer les entrées utilisateur, en les plaçant en minuscules et en informant l’utilisateur si l’option n’est pas valide.
  • À la fin de chaque tour, le joueur doit indiquer s’il souhaite rejouer ou non.

En utilisant Visual Studio Code dans votre navigateur et GitHub Copilot pour vous aider, créez un mini-jeu qui répond aux spécifications ci-dessus.

Vérifier votre travail

  1. Exécutez le mini-jeu sur la console avec la commande python app.py.
  2. À l’invite, tapez l’une des options de jeu : rock, paper ou scissors.
  3. Le mini-jeu doit indiquer au joueur s’il a gagné, a perdu ou est à égalité avec son adversaire.
  4. Choisissez de continuer à jouer.
  5. À l’invite, tapez screen.
  6. Le mini-jeu doit informer le joueur si l’option qu’il a entrée n’est pas valide.
  7. Répétez les étapes 2 et 4 pour jouer quelques tours, puis choisissez de ne pas continuer à jouer.
  8. Vérifiez si le mini-jeu est terminé et s’il affiche votre score, vous informant du nombre de victoires et de tours.

    9. Félicitations ! Vous avez réussi cet exercice de défi. Vous avez créé un mini-jeu de console en Python à l’aide de GitHub Copilot.
